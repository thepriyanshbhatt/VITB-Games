## Overview

VITB Games is an independent student-built gaming platform created to simplify access to PC games inside the college environment.

Designed specifically around campus networking limitations and user convenience, the platform delivers a smooth and modern experience for discovering, requesting, and accessing games without unnecessary complexity.

What started as a small side project evolved into a fully designed gaming platform focused on:

- Fast delivery
- Stable releases
- Affordable bundle pricing
- Premium user experience
- Curated game collections
- Campus-focused accessibility

---

# Features

## Premium Game Access
Handpicked AAA and popular PC titles optimized for stability, performance, and reliability.

## Modern Games Library
Browse categorized collections including:
- Latest Releases
- Game Bundles
- Popular Titles
- Casual & Indie Games

## Ready-To-Play Experience
Games are tested and prepared before delivery to ensure a seamless installation experience.

## Bundle Pricing
Complete game collections and franchises available at reduced pricing.

## On-Demand Requests
Users can directly request unavailable titles and custom game editions.

## Built For Campus Users
Optimized around the VIT Bhopal student ecosystem and networking environment.
